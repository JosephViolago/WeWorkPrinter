# WeWorkPrinter

## Install
- Download this repo
- Double-click! (See below for which file to click.)

### Mac OSX
- <=10.9.5 (Maverics): `printer.sh`
- 10.10+ (Yosemite): `printer.sh` [WIP]

### Linux
- `printer.sh` [WIP]

### Windows
- `printer.bat`
